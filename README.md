# Hotel Booking Form Website

## Project Description
The Hotel Booking Form is a simple web-based application that allows users to book hotel rooms by filling out a form with required details such as name, email, room type, number of guests, and booking dates. The project focuses on form validation, responsive design, and a user-friendly interface.

## Features
- Hotel booking form with required fields
- Check-in date cannot be in the past
- Check-out date must be after check-in
- Guest count validation (1–10 guests)
- Character counter for special requests (up to 500 characters)
- Helpful tooltips for user guidance
- Default check-in and check-out dates
- Fully responsive design for mobile, tablet, and desktop

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Form Validation
JavaScript validation is implemented to ensure correct input from the user.  
The system checks that all required fields are filled and verifies that the booking dates are valid before form submission.

## Responsive Design
The website uses **CSS Grid and Flexbox** to create a mobile-first layout.  
Form fields stack vertically on smaller screens and switch to a two-column layout on larger devices.
